# 《CSS世界》

# 概述

CSS世界的诞生就是为图文信息展示服务的。

我们直接称CSS为“流的世界”真是一点儿也不为过

新世界——CSS3

  • 图片元素的srcset属性，CSS的 object-fit属性
  • flex弹性
  • 栅格布局grid layout
  • 圆角、阴影和渐变
  • transform变换
  • filter滤镜
  • animation动画
